# SEO Checklist for MediaGet

1. Homepage SEO

- Title Tag: Keep under 60 characters.
  - Example: `Instagram Video Downloader - Free Reel & Photo Download`
- Meta Description: Keep around 150–160 characters.
- H1 Tag: Only one H1 per page. Example: `Instagram Video Downloader`

2. Create Separate Landing Pages

Create separate pages for each major downloader with unique title, description, FAQ and how-to:

`/instagram-video-downloader`
`/instagram-reel-downloader`
`/instagram-story-downloader`
`/tiktok-downloader`
`/youtube-shorts-downloader`
`/facebook-video-downloader`

3. URL Structure

- Use clean URLs: `example.com/instagram-video-downloader`
- Avoid query-based URLs for primary pages: `?id=123`

4. Internal Linking

- Link related downloader pages to each other to help discovery and spread link equity.

5. Images

- Every image must have descriptive `alt` text.
- Prefer WebP where supported.

6. FAQ Section

- Add an FAQ on each landing page. Consider adding FAQ Schema (JSON-LD).

7. Structured Data (Schema)

- Add: FAQ Schema, Website Schema, Breadcrumb Schema where applicable.

8. Mobile Optimization

- Touch targets large enough, readable text, no horizontal scroll, fast mobile load.

9. Speed Optimization

- Target page size < 2 MB, minify CSS/JS, lazy-load images, optimize fonts. Test with PageSpeed Insights.

10. Trust Pages

- Include: `/about-us`, `/contact`, `/privacy-policy`, `/terms-of-service`, `/dmca`.

11. Sitemap

- Provide `/sitemap.xml` listing all public pages.

12. Robots.txt

- Example:

```
User-agent: *
Allow: /

Sitemap: https://example.com/sitemap.xml
```

13. Content

- Each landing page should include 500–1000 words describing how the downloader works, supported formats/platforms, privacy, and FAQ.

14. Conversion & Above-the-Fold

- Headline, subheadline, input box, and CTA visible without scrolling.

15. Pre-launch SEO Audit

- ✅ Title on every page
- ✅ Meta description on every page
- ✅ One H1 per page
- ✅ Sitemap generated
- ✅ Robots.txt created
- ✅ Mobile responsive
- ✅ No broken links
- ✅ HTTPS
- ✅ FAQ section
- ✅ Privacy Policy
- ✅ Contact Page
- ✅ PageSpeed score 90+
